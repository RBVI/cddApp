CDDApp is a Cytoscape 3 app allowing the user to get and display domain information for proteins using the Conserved Domains Database. The domain information is stored in the nodes table and displayed through the Results panel. If the PDB identifier of the protein(s) is also available and structureViz is also installed, it will also open up the PDB structure of the protein(s) using Chimera via structureViz.

1. Installation

  1. If you haven't already done so, download and install UCSF Chimera.
  2. Install CDDApp from the Cytoscape Apps Manager (see the Apps Manager option in the Apps menu; CDDApp is listed under the Analysis Plugins category).
  3. Also install structureViz through the Apps menu if it is not in Cytoscape.
  4. Now CDDApp actions should be available in the Cytoscape Apps menu and network and node context menus.
[Contents] [Top]

2. The Open Structure Panel

The Open Structure Panel is a panel that allows the user to open the protein structure of a node if it has a PDB ID. For this feature to work, structureViz must first be installed. Each node has a sub-panel in Open Structure Panel, each sub-panel contains a drop-down list to select the protein (in the instance there is more than one protein for the node), and a button Open Structure that opens the structure in Chimera. Once the structure is opened, a list will appear in the sub-panel containing domains from CDD. Clicking on the item of the list will highlight the domain in Chimera. Click on the Close Structure button closes the structure in Chimera.

3. CDD Panel

The CDD Panel is in the Results Panel of Cytoscape, which can be open through the top-level menu View→Show Results Panel. The CDD Panel is filled by selecting one or more nodes in the network. For each node, domain information appears in the following format:

  • Node: The name of the node
  • Protein: The name of the protein
  • Table: A table containing domain information, listed in three columns:
    • Domain Name: Identifier of the domain. For domains curated by CDD, the identifier can be clicked to open up a link to get more information about the domain.
    • Domain Type: Type of domain.
    • Domain Range: Residues in the domain.

4. CDDApp menus

CDDApp provides two similar sets of menus, one under the Apps→cddApp top-level menu and the other as part of the context menus accessed by mouse right-click in the Cytoscape network or nodes.

4.1 Top-level menus

Load CDD Domains for Network
Load domain information from CDD for every node in the network. The user should choose the column to use as protein identifiers. The identifiers can be GI, uniprot or PDB. Multiple identifiers can be specified, and can be in a comma-separated String or a List of String(s). (Note: downloading information from CDD can take a long time, if you only need to load CDD information for a select number of nodes use Load CDD Domains for selected Node(s).
Load CDD Domains for selected Node(s)
Similar to Load CDD Domains for Network, except only nodes that have been selected will be loaded.
Open Structure Panel
Opens Structure Panel.

4.2 Context menus

Cytoscape context menus can be activated by right-clicking on the network. The network context menu contains Load CDD Domains for selected Node(s), which functions exactly the same way as the equivalent menu item in the top-level menu. The node context menu contains the item Load CDD Domains for Node, which functions exactly the same as Load CDD Domains for selected Node(s) but does this for the single node being clicked on.

5. CDDApp Commands

In order to facilitate the use of CDDApp, the CDDApp app exports a number of commands. These commands may be used by other apps, or as part of scripts.

cddApp loadCDDDomains4network

Loads domain information for all nodes in network from CDD.

Arguments:
  • loadColumn=ListSingleSelection: The column to use as identifiers for querying CDD.
cddApp loadCDDDomains4selectedNodes

Loads domain information for selected nodes in network from CDD.

Arguments:
  • loadColumn=ListSingleSelection: The column to use as identifiers for querying CDD.
cddApp structurePanel

Open Open Structure Panel.
